
##   حول تطبيق Taxation System 
    تطبيق بسيط للادارة الضريبية والمكلفين 
    صمم باستخدام Laravel 12 & Filament 4.3 
     
## لوحة تحكم Admin Panel
      *  تسجيل المكلفين
      *  التحققات الضريبية
      *  تسجيل المدفوعات
      * براءات الذمة  / استعلام - عرض - طباعة  
      * صفحة للاحصائيات
      * عبر الرابط       http://localhost:8000/admin/login 

## استعلام خارجي Public Panel  
      * صفحة للاستعلام الخارجي للمكلفين باستخدام الرقم الضريبي
      * عبر الرابط    http://localhost:8000/public/check-debt
